Secure Medical RAG Pipeline

github.com/sammcdsam/medical-rag-pipeline

Agentic, access-controlled, federated RAG over live PubMed — air-gap-capable, exposed over MCP · Python

  • Built an agentic RAG loop (Claude) over 27K live PubMed abstracts (34K chunks) that decomposes multi-hop questions and reformulates its own queries — with the caller's clearance bound so the agent cannot escalate its own access, making retrieval resistant to prompt injection from retrieved text.
